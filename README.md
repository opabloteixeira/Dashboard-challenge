## :information_source: Como usar

```bash

# Clone este repositório
$ git clone https://github.com/opabloteixeira/go-finances-front

# Vá até p repositório
$ cd go-finances-front

# Instale as dependências
$ yarn

# Rode a aplicação
$ yarn start

```
